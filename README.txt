• Repetitive Nearest Neighbor Search AI agent
• Designed with use of a Pac-Man video game library
• Helps Pac-Man to find all food pellets on any given course using RNNA method

To run this Pac-Man RNNA agent
1) Open a command prompt/terminal 
2) Navigate to the folder that contains this repository  
3) type the following:
	java -jar PacSimRNNA.jar MAP
	
	where MAP = the desired map for pacman to navigate.
	Either tsp-tiny or tsp-original-sparse20
